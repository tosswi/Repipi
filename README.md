# アプリ名

Pepipi

## サイト概要

Repipi とは料理好きのための SNS です。

### サイトテーマ

この人が作っている料理を真似をしたい！この人からアドバイスをもらいたい！といった料理好きなら誰もが思うことを形するサイトです。

### ターゲットユーザ

料理好きな人はもちろん、料理を始めたいと思う人がアドバイスを求めやすくするため、チャット機能を搭載いたしました。
